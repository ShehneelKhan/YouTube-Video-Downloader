# YouTube-Video-Downloader
It is a YouTube Video Downloader made with Python and PyQt5 library.

## Necessary Dependencies
* Python
* QtDesigner

## Necessary Packages
* PyQt5
* pytube

## How to use
Just copy url of the youtube video which you want to download and paste it in the application. Click "Download" then wait for a while (1 minute approx) and the video will be downloaded on the folder from which you are using this program.

## Note
It is possible that this program does not work on your computer because youtube continously change their algorithms which will reflect upon this program. So, if this happens with you, you might have to change the files which are inside the pytube library by taking some help from google or github.
